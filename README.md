# node-dojo-location
Excercise to start practising with node.js

## Introduction
Given an IP address return the CountryCode

## Getting started
Installation:

```bash
git clone https://github.com/manuelmorales/location.git
cd location
npm install
grunt bootstrap # Will download the database
```

Check that everything is in order:

```bash
grunt test
```


Run the server:

```bash
node app.js
```
